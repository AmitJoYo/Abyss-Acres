# 🧠 Agent Memory — Abyss & Acres

## Project Summary
**Abyss & Acres** is an infinite, torus-wrapping Snake.io clone built in **Godot 4.x (GDScript)** targeting Android (60 FPS, offline, ad-free). Two themes: "The Meadow" (farm) and "The Abyss" (deep sea bioluminescence).

## Key Architecture Decisions
1. **Torus wrapping** uses `wrap_delta()` (shortest-path delta) not absolute positions — segments never know a teleport happened
2. **Ghost rendering** — duplicate sprites drawn at `pos ± WORLD_SIZE` for any node within 600px of edge
3. **Object pooling** for segments (200 pre-alloc) and food (120 pre-alloc)
4. **PointLight2D** limited to ~15 on-screen (cull off-screen bot lights) for Android perf
5. **ThemeData** is a Godot Resource (.tres) — all art/audio/shader refs in one swappable file
6. **All distance/collision checks** use `wrapped_distance()` — works correctly near edges

## How to Continue
1. Open project in Godot 4.x → Import `project.godot`
2. Install GdUnit4 addon for tests
3. Generate art using prompts in `Docs/06_UI_Art_Prompts.md`
4. Place textures in `Assets/Textures/Meadow/` and `Assets/Textures/Abyss/`
5. Create `Resources/ThemeMeadow.tres` and `Resources/ThemeAbyss.tres` using ThemeData class
6. Build Sprint 4: Game Over UI, skin select, score polish
7. Build Sprint 5: Android export, QA
